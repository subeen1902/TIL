## Markdown

- text 기반의 경량 마크업 언어 ⇒ `.md` 확장자 사용
- 주요 사용처 : GitHub, Notion, Obsidian, 개발 문서 등

| 문법 요소      | 기호 / 형식                     |
| -------------- | ------------------------------- |
| 제목           | `#`, `##`, `###`, ..., `######` |
| 굵은 글씨      | `**굵게**` 또는 `__굵게__`      |
| 기울임         | `*기울임*` 또는 `_기울임_`      |
| 취소선         | `~~취소선~~`                    |
| 인용           | `> 인용문`                      |
| 순서 없는 목록 | `- 항목`, `* 항목`, `+ 항목`    |
| 순서 있는 목록 | `1. 항목`, `2. 항목`, ...       |
| 체크박스       | `- [ ] 할 일`, `- [x] 완료됨`   |
| 인라인 코드    | `` `코드` ``                    |
| 코드 블록      | <pre>`언어<br>코드<br>`</pre>   |
| 수평선         | `---`, `***`, `___`             |
| 링크           | `[텍스트](URL)`                 |
| 이미지         | `![대체 텍스트](이미지 URL)`    |
| 줄바꿈         | 줄 끝에 공백 두 칸 `␣␣`         |
| 표             | `\|` , `-` (구분선) 사용        |
