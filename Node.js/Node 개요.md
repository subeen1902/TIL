## Module, Package, Library

- 모듈(module) : 프로그램을 기능별 작은 단위로 쪼갠 것, 다른 곳에서 사용 가능하도록 파일 형태로 저장 ⇒ 재사용 가능
- 패키지(package) : 자주 사용하는 기능 모듈들의 묶음 ⇒ 배포의 단위
- 라이브러리(library) : 모듈보다 더 큰 단위, 라이브/모듈을 한꺼번에 모아서 칭하는 말
